Server Installation

The Hyperledger Server is currently in beta

See the latest instructions, code, and issues on GitHub

  1. Install Elixir
  2. Clone the repo
    git clone https://github.com/hyperledger/hyperledger-server.git
  3. Install dependencies
    mix deps.get
  4. Create a database
    mix ecto.create Hyperledger.Repo
  5. Run Migration
    mix ecto.migrate Hyperledger.Repo
  6. Start Hyperledger
    mix phoenix.server
  7. The endpoint is now running
    localhost:4000

Client Installation

The Hyperledger Client is currently in beta

See the latest instructions, code, and issues on GitHub

  1. Install Ruby
  2. Install RubyGems
  3. Install Hyperledger CLI
    gem install hyperledger_cli
  • Create Ledger
  • Issue Units
  • Create Account
  • Make Transfer
> hyperledger ledger register newasset http://www.newasset.org

New ledger created
name: newasswet
url: http://www.newasset.org
primary account code: 9d996e0...
> hyperledger issue 1000000 newasset

1000000 new units issued to newasset
> hyperledger account register newasset

New account created
code: 9dc1db...
balance: 0
> hyperledger transfer 1000 9d996e0... 9dc1db...

Transfer completed!
